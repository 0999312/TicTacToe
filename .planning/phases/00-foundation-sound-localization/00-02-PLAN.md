  <action>
    Create `autoload/audio_controller.gd` with the following structure:

    ```gdscript
    extends Node

    # Sound cache: maps "tic_tac_toe:sounds/..." string keys to preloaded AudioStream
    var _sound_cache: Dictionary = {}

    func _ready() -> void:
        _ensure_audio_buses()
        _setup_bus_routing()
        _cache_sounds()
        _subscribe_events()
        _apply_volumes()
        _start_menu_music()
    ```

    Implement each private method:

    **`_ensure_audio_buses()`** (D-14/FND-03):
    - Define `var required_buses := ["SFX", "Music"]` (D-05: 2 buses under Master)
    - For each bus_name in required_buses:
      - If `AudioServer.get_bus_index(bus_name) == -1`:
        - `AudioServer.add_bus()` (appends at end)
        - `var idx := AudioServer.bus_count - 1`
        - `AudioServer.set_bus_name(idx, bus_name)`
        - `AudioServer.set_bus_send(idx, "Master")`
        - `push_warning("AudioController: created missing bus '%s' at runtime" % bus_name)`
      - Else: `print("AudioController: bus '%s' found" % bus_name)`

    **`_setup_bus_routing()`** (D-09):
    - `SoundManager.set_default_sound_bus("SFX")` -- game SFX to SFX bus
    - `SoundManager.set_default_ui_sound_bus("SFX")` -- UI sounds to SFX bus (D-05: no separate UI bus)
    - `SoundManager.set_default_music_bus("Music")` -- music to Music bus

    **`_cache_sounds()`** (D-07, D-08):
    - Create a SoundRegistry instance and register with RegistryManager:
      ```gdscript
      var registry := SoundRegistry.new()
      RegistryManager.register_registry("sound", registry)
      ```
    - Note: SoundRegistry is defined as an inner class or separate script. For simplicity, define it AS a separate script file OR as a local inner class. The D-07 requirement says "SoundRegistry extends RegistryBase" -- create a new script `res://resources/sound/sound_registry.gd`:
      ```gdscript
      extends RegistryBase
      class_name SoundRegistry

      func _validate_entry(entry: Variant) -> bool:
          return entry is AudioStream
      ```
    - Register each sound using ResourceLocation keys under `tic_tac_toe:` namespace:
      - `tic_tac_toe:sounds/game/place_mark` -> `res://addons/kenney_interface_sounds/drop_001.wav`
      - `tic_tac_toe:sounds/game/win` -> `res://addons/kenney_interface_sounds/confirmation_001.wav`
      - `tic_tac_toe:sounds/game/lose` -> `res://addons/kenney_interface_sounds/error_001.wav`
      - `tic_tac_toe:sounds/game/draw` -> `res://addons/kenney_interface_sounds/bong_001.wav`
      - `tic_tac_toe:sounds/ui/click` -> `res://addons/kenney_interface_sounds/click_001.wav`
      - `tic_tac_toe:sounds/ui/hover` -> `res://addons/kenney_interface_sounds/tick_001.wav`
      - `tic_tac_toe:music/background` -> `res://music/off_to_osaka.mp3`
    - After registration, preload all sounds into `_sound_cache`:
      ```gdscript
      for key_str in registry.get_all_keys():
          var loc := ResourceLocation.from_string(key_str)
          var stream := registry.get_entry(loc) as AudioStream
          if stream:
              _sound_cache[key_str] = stream
          else:
              push_warning("AudioController: null sound for key: ", key_str)
      ```

    **`_subscribe_events()`** (D-06):
    - Subscribe to 5 events (after Plan 01, all use StringName):
      ```gdscript
      EventBus.subscribe(&"CellPlacedEvent", _on_cell_placed)
      EventBus.subscribe(&"GameWonEvent", _on_game_won)
      EventBus.subscribe(&"GameDrawEvent", _on_game_draw)
      EventBus.subscribe(&"GameStartedEvent", _on_game_started)
      EventBus.subscribe(&"SettingChangedEvent", _on_setting_changed)
      ```

    **`_apply_volumes()`** (D-12, D-20):
    - Read volumes from SettingsManager (must exist because SettingsManager autoload order precedes AudioController):
      ```gdscript
      var sfx_vol := SettingsManager.get("audio/sfx_volume", 0.8) as float
      var music_vol := SettingsManager.get("audio/music_volume", 0.8) as float
      SoundManager.set_sound_volume(clampf(sfx_vol, 0.0, 1.0))
      SoundManager.set_music_volume(clampf(music_vol, 0.0, 1.0))
      ```
    - Clamp values to 0.0-1.0 before passing to SoundManager (mitigates T-00-03, T-00-02).

## Threat Model

### Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SettingsManager -> AudioController | Volume values cross from ConfigFile through SettingsManager to AudioServer bus levels |
| AudioServer bus creation | Runtime fallback creates buses if editor setup was missed |
| Sound file load | Kenney WAV files could be missing or corrupt on disk |

### STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-00-04 | Tampering | AudioController._apply_volumes() | mitigate | `clampf(value, 0.0, 1.0)` before passing to `SoundManager.set_sound_volume()` / `set_music_volume()` |
| T-00-05 | Denial of Service | AudioController._ensure_audio_buses() | mitigate | `AudioServer.get_bus_index()` check before `set_bus_name()`; silent fallback creates missing buses with `push_warning()` |
| T-00-06 | Denial of Service | AudioController._cache_sounds() | mitigate | `registry.get_entry()` returns null for missing keys; null-check before `_sound_cache[key_str] = stream`; `push_warning()` on load failure |
| T-00-07 | Denial of Service | Event handler sound playback | mitigate | Every `_sound_cache.get()` result null-checked before calling `SoundManager.play_sound()` |
| T-00-08 | Denial of Service | MiSans-Semibold.ttf file missing | accept | Godot falls back to default font; `push_warning` is not possible for font loading failures |
