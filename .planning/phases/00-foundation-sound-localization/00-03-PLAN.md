## Threat Model

### Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Translation JSON files -> I18NManager | JSON files on disk loaded at startup; corrupt file causes load failure |
| User language selection -> TranslationServer | Language switch triggers locale change on TranslationServer |

### STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-00-09 | Denial of Service | I18NManager.load_translation() | mitigate | `load_translation()` returns `false` on file read failure or JSON parse failure (already implemented); game continues with untranslated strings |
| T-00-10 | Denial of Service | tr() key not found in current locale | accept | Godot's `tr()` falls back to the key string when no translation exists -- no crash, no error |
| T-00-11 | Tampering | main.tscn text field injection | accept | Scene files are part of the game binary (not user-modifiable) -- only editable via Godot editor |
