      ```gdscript
      @export var sfx_volume: float = 0.8

      @export var difficulty: int = 1
      @export var locale: String = "zh_CN"
      ```
    - Constants:

    - Private state:

      `var _cache: Dictionary = {}` -- key format: "section/key" -> value

    - Public API `get(key: String, default: Variant = null) -> Variant`:
      Returns `_cache.get(key, default)`. Key format "section/key" (e.g., "audio/sfx_volume", "gameplay/difficulty", "language/locale").
    - Public API `set(key: String, value: Variant) -> void`:
      Stores key-value in _cache, publishes `SettingChangedEvent.new(key, value)` via EventBus.
    - Public API `save() -> void`:
      Iterates _cache, splits key on "/", writes to ConfigFile sections. Calls `_config.save(SETTINGS_PATH)`. On error, calls `push_error("SettingsManager: failed to save settings: ", err)`.
    - Private `_load_config()`:
      Calls `_config.load(SETTINGS_PATH)`. If err != OK, calls `push_warning("SettingsManager: no settings file found (%d), using defaults" % err)` and uses default values from exported properties.
      On success, reads each known setting key from ConfigFile and populates _cache.
    - Private `_populate_cache_from_config()`:
      Reads sections: [audio] (sfx_volume, music_volume), [gameplay] (difficulty), [language] (locale).
      Falls back to exported default values for any missing keys.
    - Signal registration: `get_tree().notification(NOTIFICATION_WM_CLOSE_REQUEST)` -> call save() on game exit, then `get_tree().quit()`.
