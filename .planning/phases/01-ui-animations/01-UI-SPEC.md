# Phase 1 -- UI Design Contract

> Visual and interaction contract for Phase 1 (UI and Animations). Defines the Godot-specific design tokens, panel architecture, animation specifications, and copywriting keys. Generated by gsd-ui-researcher.

---

## Design System (Godot)

| Property | Value |
|----------|-------|
| Engine | Godot 4.6, Forward Plus, 720x720 canvas_items |
| Theme resource | `res://resources/theme/default_theme.tres` (MiSans-Semibold, default_font_size=20) |
| Font | MiSans-Semibold (only weight available: Semibold, effectively weight 600) |
| Panel system | UIPanel base class + UIManager stack (NORMAL/POPUP layers) |
| Container strategy | VBoxContainer (vertical layouts), HBoxContainer (horizontal layouts), TabContainer (settings tabs), MarginContainer (panel padding) |
| StyleBox | StyleBoxFlat for panel backgrounds, button bg, tab bar bg; no StyleBoxTexture used |
| Icon assets | None -- all visual identity from SVG sprites (x_mark.svg, o_mark.svg, highlight_square.svg) |
| Input | GUIDE actions per panel (GUIDEMappingContext created/destroyed in _on_open/_on_close) |
| Audio integration | AudioController singleton -- panels call `AudioController.play_button_click()` / `play_button_hover()` on interaction |
| Localization | All text via `tr()` -- I18NManager handles zh_CN/en_US/ja_JP |
| Animation engine | Tween (create_tween()) on individual Control/Node2D properties |
| Viewport safe area | Full 720x720 canvas -- no notch/mobile concerns |

## Spacing Scale

Pixel-based spacing (Godot uses raw pixel values, not CSS rem). Use multiples of 4 where possible for consistency. Referenced in scene file theme_override_constants and custom_minimum_size.

| Token | Value (px) | Usage |
|-------|------------|-------|
| xs | 4 | Inline padding inside buttons, label-to-edge gaps |
| sm | 8 | Compact element spacing, icon-to-text gap |
| md | 16 | Default separation between stacked elements |
| lg | 24 | Section padding, panel interior margin |
| xl | 32 | Between major UI blocks (title-to-buttons gap) |
| 2xl | 48 | Page-level section breaks |
| 3xl | 64 | Outer panel margins, max separation |

**Panel-specific spacing (theme_override_constants/separation):**
- MainMenu VBox: separation=20
- GameOver VBox: separation=15
- HUD VBox: separation=8 (compact score display)
- Settings TabContainer tabs: separation=16
- PauseMenu VBox: separation=20

**Button minimum size (custom_minimum_size):**
- Menu buttons: width=260, height=48 (comfortable touch target)
- Settings buttons: width=200, height=40
- Game over buttons: width=160, height=40

**Exceptions:** Grid cells have 200px spacing (160px cell + 40px gap between centers), determined by the board coordinate math, not the spacing scale.

## Color

Color values use `Color(r, g, b, a)` format for Godot. Hex equivalents provided for reference.

| Role | Godot Color | Hex | Usage |
|------|-------------|-----|-------|
| Dominant surface (60%) | `Color(0.08, 0.08, 0.12, 1)` | `#14141F` | Main background, canvas behind board, panel backgrounds |
| Surface card (30%) | `Color(0.14, 0.14, 0.22, 1)` | `#242438` | Panel interior surfaces, button default bg, Settings tab content area |
| Surface elevated | `Color(0.20, 0.20, 0.30, 1)` | `#33334D` | Button hover state bg, selected tab, PauseMenu overlay card |
| Accent (10%) | `Color(0.91, 0.72, 0.29, 1)` | `#E8B84B` | Win line, cursor highlight pulse, selected difficulty button, button hover brighten additive |
| Text primary | `Color(0.95, 0.95, 0.98, 1)` | `#F2F2FA` | All labels and button text (default modulate) |
| Text secondary | `Color(0.60, 0.60, 0.70, 1)` | `#9999B3` | Version number, score numbers, subtitle text |
| Grid lines | `Color(0.70, 0.70, 0.70, 1)` | `#B3B3B3` | Board grid lines (existing) |
| Dimmer overlay | `Color(0, 0, 0, 0.5)` | `#00000080` | UIManager background dimmer, pause dimmer |
| X mark | `Color(0.31, 0.76, 0.97, 1)` | `#4FC3F7` | X sprite modulate (existing from SVG fill) |
| O mark | `Color(1.00, 0.54, 0.40, 1)` | `#FF8A65` | O sprite modulate (existing from SVG fill) |
| Destructive | `Color(1.00, 0.27, 0.27, 1)` | `#FF4444` | Quit button text/hover (only on quit button) |

**Accent reserved for:** Win line Tween draw, cursor highlight pulse color, button hover brighten (modulate * 1.15), selected difficulty button highlight, actively selected tab indicator. NOT used for: X/O marks (fixed blue/orange), grid lines, background fills.

**Button color convention:**
- Default bg: `#242438` with modulate = Color(1, 1, 1, 1)
- Hover: modulate = Color(1.15, 1.15, 1.15, 1) -- achieved via Tween from 1.0 to 1.15 over 0.1s (brightening)
- Press: modulate = Color(0.95, 0.95, 0.95, 1) -- achieved via Tween from current to 0.95 over 0.1s (darkening)
- No separate hover/press texture -- all via modulate + scale transform
- Disabled: modulate = Color(0.5, 0.5, 0.5, 0.6) (not used in this phase, reserved)

**TabBar button style (Settings panel):**
- Unselected tab: `#242438` bg, text = Text primary
- Selected tab: `#33334D` bg, underline accent color bar (StyleBoxFlat.border_color = accent, border_width.bottom = 2)

## Animation Tokens

All animations use `create_tween()` on the target node. Tween instances are stored as `@onready var _tween: Tween` and killed before starting a new one to prevent overlap.

| Animation | Target Property | Start | End | Duration (s) | Easing / Trans | Notes |
|-----------|----------------|-------|-----|--------------|----------------|-------|
| Place mark (scale in) | sprite.scale | Vector2(0, 0) | Vector2(1.2, 1.2) then Vector2(1.0, 1.0) | 0.25 | Trans.BOUNCE, Ease.OUT | Two-step Tween: scale to 1.2 then bounce back to 1.0 |
| Win line draw | Line2D.width | 0 | 6.0 | 0.3 | Trans.SPRING, Ease.OUT | Tween default_color alpha from 0 to 1 simultaneously |
| Cell highlight pulse | Sprite2D.modulate.a | hidden | 0.6 opacity | 0.4 (loop) | Trans.SINE, Ease.IN_OUT | Use parallel Tween loop on winning cells' highlight sprite. Pulse repeats 2 times then stops. |
| Scene fade out | ColorRect.modulate.a | 1.0 | 0.0 | 0.3 | Trans.LINEAR | On fade out: tween alpha to 0, then change scene, then fade in |
| Scene fade in | ColorRect.modulate.a | 0.0 | 1.0 | 0.3 | Trans.LINEAR | On fade in: alpha starts at 0, tweens to 1 after scene loaded |
| Button hover (scale) | Control.scale | Vector2(1.0, 1.0) | Vector2(1.05, 1.05) | 0.1 | Trans.SINE, Ease.OUT | Only if not already hovered (guard against re-trigger) |
| Button hover (brighten) | Control.modulate | Color(1,1,1,1) | Color(1.15,1.15,1.15,1) | 0.1 | Trans.SINE, Ease.OUT | Run parallel with scale tween |
| Button press | Control.scale | Vector2(1.0, 1.0) | Vector2(0.95, 0.95) | 0.1 | Trans.SINE, Ease.OUT | On button_down, not on pressed (instant feel) |
| Button release | Control.scale | Vector2(0.95, 0.95) | Vector2(1.0, 1.0) | 0.1 | Trans.SINE, Ease.OUT | On button_up |
| Music crossfade | (SoundManager.play_music crossfade param) | current track | new track | 1.0-1.5 | N/A (SoundManager API) | Menu to gameplay: off_to_osaka -> breaktime. Gameplay to menu: breaktime -> off_to_osaka. |
| Pause music lower | SoundManager volume | current music_volume | current * 0.5 | 0.3 | Trans.SINE, Ease.OUT | Only if current music_volume > 0.1 |
| Resume music restore | SoundManager volume | lowered volume | original volume | 0.3 | Trans.SINE, Ease.OUT | On PauseMenu close |

**Tween lifecycle rules:**
1. Kill previous tween before starting new one on same node/property
2. Use `set_parallel(true)` for simultaneous scale + modulate on buttons
3. Win line Tween: run as chain -- first draw line, then cell pulse (or parallel if visually acceptable)
4. Scene transition ColorRect must be a child of the root CanvasLayer with highest layer (SYSTEM=400 equivalent) to ensure it always renders on top

## Layout Contracts

### Panel Architecture

Each panel is an independent `.tscn` scene with a root node extending `UIPanel`. Registered with UIManager at startup.

| Panel | Scene Path | Layer | Cache Mode | ResourceLocation ID | Opened By |
|-------|-----------|-------|------------|---------------------|-----------|
| MainMenu | `res://scenes/panels/main_menu.tscn` | NORMAL (100) | CACHE | `tic_tac_toe:main_menu` | On app start (from main.gd _ready) |
| HUD | `res://scenes/panels/hud.tscn` | NORMAL (100) | CACHE | `tic_tac_toe:hud` | Game start (on GameStartedEvent) |
| GameOver | `res://scenes/panels/game_over.tscn` | POPUP (200) | NONE | `tic_tac_toe:game_over` | Win/draw event |
| PauseMenu | `res://scenes/panels/pause_menu.tscn` | POPUP (200) | NONE | `tic_tac_toe:pause_menu` | Esc during gameplay (GUIDE action) |
| Settings | `res://scenes/panels/settings.tscn` | NORMAL (100) | CACHE | `tic_tac_toe:settings` | MainMenu Settings button OR PauseMenu Settings button |

### Navigation Flow

```
App Start
  -> UIManager.open_panel("tic_tac_toe:main_menu")  (NORMAL layer)

MainMenu - [双人对战/人机对战] pressed
  -> UIManager.close_panel("tic_tac_toe:main_menu")
  -> UIManager.open_panel("tic_tac_toe:hud")         (NORMAL layer)
  -> Board scene instantiated to scene tree
  -> Music crossfade: off_to_osaka -> breaktime

GameOver triggered (win/draw)
  -> UIManager.open_panel("tic_tac_toe:game_over")   (POPUP layer, overlays HUD)

GameOver - [再来一局] pressed
  -> GameManager.reset_board()
  -> UIManager.close_panel("tic_tac_toe:game_over")   (POPUP closes, HUD resumes)

GameOver - [返回主菜单] pressed
  -> UIManager.close_all(NORMAL)                      (closes both NORMAL panels)
  -> Board scene removed from tree
  -> UIManager.open_panel("tic_tac_toe:main_menu")
  -> Music crossfade: breaktime -> off_to_osaka

Esc during gameplay
  -> UIManager.open_panel("tic_tac_toe:pause_menu")   (POPUP layer, with dimmer)
  -> Music volume reduced by 50%

PauseMenu - [继续游戏] pressed
  -> UIManager.back(POPUP)
  -> Music volume restored

PauseMenu - [设置] pressed
  -> UIManager.open_panel("tic_tac_toe:settings")     (NORMAL layer, paused HUD underneath)

PauseMenu - [返回主菜单] pressed
  -> UIManager.close_all()
  -> Board scene removed from tree
  -> UIManager.open_panel("tic_tac_toe:main_menu")
  -> Music crossfade: breaktime -> off_to_osaka

Settings [返回] pressed (from any context)
  -> UIManager.back(NORMAL)                            (pops to previous NORMAL panel)

MainMenu - [退出游戏] pressed
  -> get_tree().quit()
```

## Code Quality Contracts

**CQ-02 (Parameter validation):**
Add `push_warning` to GameManager public methods for invalid parameter values:
- `GameManager.place_mark(cell_index)` -- warn if cell_index < 0 or cell_index > 8
- `GameManager.move_cursor(direction)` -- warn if direction is zero vector
- `GameManager.start_game(mode, player_side)` -- warn if invalid mode or player_side enum values

**CQ-03 (Duplicate code extraction):**
Extract button hover/press animation logic into a shared helper:
```gdscript
# In a shared script (e.g., scripts/utils/ui_helpers.gd or a method on panels)
func _setup_button_animation(button: Button) -> void
```
This helper handles the hover scale+modulate tween and press scale tween for any button. All panels use this single helper rather than duplicating Tween setup.
